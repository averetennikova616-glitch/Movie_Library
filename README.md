Описание программы «Movie Library»
Автор: Арина Веретенникова

Краткое описание
Программа «Movie Library» — это удобное графическое приложение на Python для хранения, просмотра и фильтрации кинематографической коллекции. Пользователь может добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Также есть возможность фильтровать список по жанру и году, а все данные сохраняются автоматически в файл, что позволяет сохранять вашу коллекцию между сессиями.

Инструкция по запуску
Требуемые зависимости
Для работы программы понадобится установленный Python версии не ниже 3.6.
Дополнительные библиотеки, используемые в программе:

tkinter — обычно входит в стандартную поставку Python (для Windows и macOS).
json — также входит в стандартную библиотеку Python.
Если у вас по каким-то причинам tkinter не установлен, на Windows его обычно можно активировать через установщик или через менеджер пакетов вашей ОС. В Linux можно установить командой:

bash

sudo apt-get install python3-tk
Запуск программы
Скачайте весь код программы в файл, например, movie_library.py.
Откройте командную строку или терминал и перейдите в директорию с файлом:
bash

cd путь_к_вашей_папке
Запустите программу командой:
bash

python3 movie_library.py
Примеры использования и работы программы
Добавление фильма:
Введите в поля «Название», «Жанр», «Год выпуска» и «Рейтинг» (от 0 до 10), затем нажмите кнопку «Добавить фильм». Фильм появится в списке и сохранится в файл.

Фильтрация по жанру:
Выберите в выпадающем списке «Фильтр по жанру» нужный жанр или «Все» для отмены фильтрации. Список фильмов обновится автоматически.

Фильтрация по году:
Выберите в выпадающем списке «Фильтр по году» нужный год или «Все», чтобы просматривать фильмы определенного года.

Просмотр и редактирование коллекции:
Все добавленные фильмы отображаются в таблице. Чтобы добавить новый фильм, заполните поля и нажмите «Добавить фильм». Данные сохраняются автоматически.
